В рамках практической работы было разработано Android-приложение на Kotlin, в котором весь интерфейс создаётся программно без изменения XML-файлов. В коде формируется вертикальный скроллируемый макет с TextView “Hello Programmed-View!”, рядом из десяти TextView с цифрами 0–9, десятком кнопок с обработчиками нажатий, а также двумя вариантами калькулятора на GridLayout: один собран вручную, второй — циклом по списку клавиш. Логика калькулятора реализована через парсер математических выражений со стеками значений и операторов. Стилизация всех TextView и кнопок выполняется циклом с использованием GradientDrawable и параметров отступов, что демонстрирует программное применение единого стиля ко всем элементам интерфейса.
